# Dataflow Gen2 Design

## Overview

Dataflow Gen2 transforms Bronze datasets into trusted Silver datasets using low-code data preparation.

---

## Responsibilities

- Data type correction
- Data quality validation
- Standardization
- Business enrichment
- Silver table publication

---

## Input

Bronze Tables

---

## Output

Silver Tables

---

## Design Principles

- Low-code transformations
- Reusable business logic
- Data quality first
- Separation of concerns
